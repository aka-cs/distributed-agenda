# Indicaciones para compilar y ejecutar el proyecto

## Requisitos

* **Python**: [Python 3.10](https://www.python.org/downloads/)
* **Golang**: [Go 1.18](https://golang.org/dl/)

## Instalación de dependencias

```bash
make client-install
make server-install
```

## Ejecución

```bash
make client
make server
```
